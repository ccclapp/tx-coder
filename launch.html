QBO Coding Agent

Automated Bank Transaction Coding for QuickBooks Online

Private Application

About This Application

QBO Coding Agent is a private, internal-use application developed by Cushing Development Corp, Inc. for automated classification and coding of bank feed transactions across multiple QuickBooks Online entities.

This application is not available for public use. It is authorized exclusively for QuickBooks Online companies owned and managed by Cushing Development Corp, Inc. and its related parties.

What This Application Does

  • Reads uncoded bank feed transactions from QuickBooks Online
  • Classifies transactions using rules-based matching and AI-assisted categorization
  • Maps transactions to the correct accounts and vendors
  • Writes coded transactions back to QuickBooks Online for review

QuickBooks Online Connection

This application connects to QuickBooks Online using Intuit's OAuth 2.0 authorization framework. The connection requires explicit authorization from a QBO administrator and can be revoked at any time.

Data access is limited to: transactions, chart of accounts, vendor lists, and company information. No personal employee data, payroll, or customer personal information is accessed.
