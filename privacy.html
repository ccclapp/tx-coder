This Privacy Policy describes how Cushing Development Corp, Inc. ("Company," "we," "us") collects, uses, stores, and protects data accessed through the QBO Coding Agent application ("Application") in connection with Intuit QuickBooks Online ("QBO").

1. Data We Access

The Application accesses the following types of data from QuickBooks Online through Intuit's authorized APIs:

Data TypePurpose
Bank feed transactionsTransaction classification and coding
Chart of accountsAccount mapping for transaction coding
Vendor listsVendor matching and categorization
Purchase and bill recordsHistorical pattern analysis for coding accuracy
Company informationEntity identification
TransactionsEnter, edit, and delete to achieve intended accounting outcomes
ReportsVerify coding and transaction accuracy

We do not access personal employee data, payroll information, customer personal information, Social Security numbers, or bank account credentials.

2. How We Use Data

QBO data is used exclusively to:

  • Classify and code bank feed transactions for accounting purposes
  • Match transactions to existing vendors and accounts
  • Enter, edit, and delete transactions to achieve intended accounting outcomes
  • Write coded transactions back to QuickBooks Online
  • Improve coding accuracy through historical pattern analysis

We do not use QBO data for marketing, advertising, profiling, or any purpose unrelated to accounting operations.

3. Data Storage and Retention

  • Transit: Data is transmitted over encrypted HTTPS connections using TLS 1.2 or higher.
  • At rest: Transaction data may be temporarily cached locally for processing. Cached data is stored on encrypted storage and is deleted after processing is complete or within 30 days, whichever comes first.
  • OAuth tokens: Access and refresh tokens are stored in encrypted configuration files and are never exposed in logs, URLs, or client-side code.
  • No permanent storage: We do not maintain a permanent database of QBO financial data. QuickBooks Online remains the system of record.

4. Data Sharing

We do not sell, rent, lease, or share QBO data with any third parties. Data is accessed and processed solely by the Company for internal accounting operations.

The Application connects to QBO through CData Connect AI, a certified data integration platform. CData acts as a pass-through connector and does not store QBO data beyond the scope of active API requests.

5. Data Security

We implement the following security measures:

  • OAuth 2.0 authorization with CSRF state parameter validation
  • Encrypted token storage (tokens never stored in plaintext)
  • HTTPS-only communication with TLS 1.2+
  • Access limited to authorized personnel of Cushing Development Corp, Inc.
  • No client secrets or credentials exposed in frontend code or logs

6. Your Rights

Authorized users may:

  • Revoke access at any time by disconnecting the Application through QBO's "Manage Apps" settings or by contacting us
  • Request data deletion by emailing caleb@cushingco.net — we will delete all cached data within 15 business days
  • Request information about what data has been accessed by contacting us

7. Children's Privacy

The Application is a business accounting tool and is not directed at individuals under the age of 18. We do not knowingly collect data from minors.

8. California Privacy Rights (CCPA/CPRA)

If you are a California resident, you have the right to:

  • Know what personal information is collected and how it is used
  • Request deletion of personal information
  • Opt out of the sale of personal information (we do not sell any data)

To exercise these rights, contact caleb@cushingco.net.

9. International Data Transfers

All data processing occurs within the United States. We do not transfer QBO data outside of the United States.
